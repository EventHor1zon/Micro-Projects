Basic RTC clock program for the STM32

Written using STM32 HAL libs.

Hardware: 1x Common Cathode 4 * 7-segment displays
          4x Transistors
          1x DS3231 RTC breakout
          1x 74HC595 Shift register
          4x 1KOhm res, 4x 330Ohm res
